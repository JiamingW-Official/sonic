# sonic

**Sound Matrix** — Web Audio 键盘合成器 + 鼓组 + 手势/头部追踪。

- **应用目录**: [Sound_Interactions/](Sound_Interactions/)
- **本地运行**: `cd Sound_Interactions && npm run dev` → http://localhost:3000
- **GitHub Pages**: 使用 **GitHub Actions** 部署（见 `.github/workflows/deploy-pages.yml`）。  
  在仓库 **Settings → Pages → Build and deployment → Source** 中选择 **GitHub Actions**，推送后会自动把 `Sound_Interactions` 发布为站点根目录。
